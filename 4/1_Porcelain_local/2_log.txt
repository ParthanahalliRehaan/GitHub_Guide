## ðŸ”¹ 1. Basic Usage
```bash
git log
```
**Purpose**:  
- Displays the **commit history** of your repository.  
- Each entry shows commit ID (SHA), author, date, and message.  
- Default view is verbose and chronological (newest first).

## ðŸ”¹ 3. Common Options

| Flag | Meaning | Example |
|------|---------|---------|
| `--oneline` | Compact view (SHA + message) | `git log --oneline` |
| `--graph` | ASCII graph of branches/merges | `git log --graph` |
| `--decorate` | Show branch/tag names | `git log --decorate` |
| `--stat` | Show file changes summary | `git log --stat` |
| `-p` | Show patch (diff) for each commit | `git log -p` |
| `--name-only` | Show only filenames changed | `git log --name-only` |
| `--name-status` | Show filenames + status (A/M/D) | `git log --name-status` |

### Show Diffs
```bash

```
- Displays the actual code changes for each commit.  
- Useful for reviewing history in detail.

## ðŸ”¹ 4. Filtering & Searching

### Limit Number of Commits
```bash
git log -n 5
```
- Show only last 5 commits.

### Filter by Author
```bash
git log --author="Rehaan"
```

### Filter by Date
```bash
git log --since="2026-05-01" --until="2026-05-25"
```

### Filter by Message
```bash
git log --grep="feature"
```

### Filter by Path
```bash
git log -- src/app.js
```
- Show commits affecting a specific file.

## ðŸ”¹ 5. Pretty Formats

### Custom Format
```bash
git log --pretty=format:"%h - %an, %ar : %s"
```
Output:
```
1a2b3c4 - Rehaan, 2 hours ago : Add login feature
9z8y7x6 - Rehaan, 1 day ago : Fix typo
```

**Common placeholders**:
- `%h` â†’ short commit hash  
- `%an` â†’ author name  
- `%ar` â†’ relative date  
- `%s` â†’ subject (commit message)  

---

### Predefined Formats
- `--pretty=oneline` â†’ one commit per line.  
- `--pretty=short` â†’ short author + message.  
- `--pretty=full` â†’ full author + committer info.  
- `--pretty=fuller` â†’ even more metadata.

## ðŸ”¹ 6. Advanced Visualizations

### Show Branch History
```bash
git log --oneline --graph --all
```
- Shows all branches in one graph.  

### Show Reflog (local HEAD movements)
```bash
git reflog
```
- Not exactly `log`, but shows HEAD changes (resets, checkouts, rebases).
